Adds a simple featured website ribbon to the homepage of your website.

== Description ==

If your website has been featured on www.admiretheweb.com you may want to show some love back. This plugin allows you to simply install and you have made a credit back to us.

* Displays badge only on your homepage

== Installation ==

1. Upload the `admire-featured-badge` folder to the `/wp-content/plugins/` directory, or search for **Admire Featured Badge** in your WordPress admin area.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Once activated you will see the badge on your homepage.
